 FAST TUTORIAL DOCKER
-docker ps : exibe as imagens ativas
-docker images : exibe as imagens geradas
-docker system prune : limpa erros imagens
-docker image rmi nomeimage:version : remove imagem
-docker-compose logs -f : exibe todos os logs dos containeres
-docker logs -f server-apache : exibe o log do container
-docker network list : exibe os dados da rede gerada pelo container
